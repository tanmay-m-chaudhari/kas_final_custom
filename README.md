# Recipe Browser

A Vue 3 recipe browsing app. Browse a curated collection of recipes and filter them by category.

## Features

- Browse all recipes with ingredient count and prep time
- Filter recipes by category (Breakfast, Lunch, Dinner, Dessert, Snack)
- Click any recipe card to view full ingredients and instructions

## Tech Stack

- Vue 3 (Composition API)
- Vite
- Static data (no backend required)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Build

```bash
npm run build
npm run preview
```
